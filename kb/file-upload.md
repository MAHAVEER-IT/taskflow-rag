# File Upload

## Overview

TaskFlow lets you upload files to projects, tasks, comments, and workspace resources so your team can keep important materials near the work they support. Uploaded files can include briefs, spreadsheets, images, PDFs, design files, meeting notes, and other project documents.

Files inherit access from the place where they are uploaded. A file attached to a private project is visible only to users who can access that project.

## Steps

1. Open the project, task, or comment where you want to add a file.
2. Select the paperclip icon or **Upload file**.
3. Choose a file from your computer.
4. Wait for the upload progress indicator to complete.
5. Add an optional description or comment.
6. Select **Attach** or **Post**.

To share an uploaded file, open the file preview and select **Copy link**. Share the link only with people who already have access to the project or task. TaskFlow checks permissions before opening file links.

To remove a file, open the file menu and select **Delete file**. Depending on your role, you may only be able to delete files you uploaded.

## Important Notes

- Supported formats include PDF, DOCX, XLSX, PPTX, CSV, PNG, JPG, GIF, TXT, ZIP, and common design files.
- Individual files can be up to 250 MB on standard plans.
- Higher plans may include larger file limits and additional storage.
- Files count toward your workspace storage limit.
- Do not upload confidential information unless your workspace policies allow it.
- Deleting a task may also remove or hide files attached only to that task.

## Troubleshooting

If an upload fails, check your internet connection and try again. Large files may take longer on slower connections.

If TaskFlow says the file is too large, compress the file, split it into smaller parts, or ask an Admin about upgrading the plan.

If the file type is not supported, place the file in a ZIP archive or use a supported format.

If teammates cannot open the file, confirm they have access to the project or task where the file was uploaded.

## FAQ

**Can I upload multiple files at once?**  
Yes. Select multiple files from your computer when the upload window opens.

**Can I preview files in TaskFlow?**  
Common images, PDFs, and office documents can usually be previewed directly in TaskFlow.

**Can I link files from Google Drive instead of uploading them?**  
Yes. Connect Google Drive from **Workspace settings > Integrations** and attach Drive files to tasks or projects.

**Who can delete uploaded files?**  
Admins, Managers, project owners, and the original uploader may delete files depending on workspace settings.
